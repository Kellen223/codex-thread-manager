# Codex Thread Manager — Windows 安装

## 安装

把下面这个链接发给 Codex：

```
https://github.com/Kellen223/codex-thread-manager
```

Codex 会自动 clone。

## 启动

clone 完成后，在 Codex 终端运行：

```
node server.js
```

然后浏览器打开 `http://127.0.0.1:8520`。

## 如果提示"node 找不到"

先安装 Node.js：打开 https://nodejs.org ，下载 LTS 版本安装。  
装完后重启 Codex，再运行 `node server.js` 即可。

## 停止

在终端按 `Ctrl + C`。
